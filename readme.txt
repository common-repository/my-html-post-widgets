A Simple Plugin to insert a Google Adsense and HTML Widgets display Under Below Post Title and Below Post Content.

== Description ==
Hai this is a Simple Plugin For Insert Adsense Code & other HTML widgets Below Post Title and Below Post Content.

This plugin Help the Bloggers to Increase their Ads and Affiliate CTR & Conversions.

- Insert Google Adsense <b>Below Post Title</b> and <b>Below Post content</b>
- Supports HTML Widgets
- it Does not Support PHP Codes
- Supported all Responsive HTML Tags

If you Have Any Questions Related in this Plugin Please Use the WordPress forum and I will take care of any issue.

== Installation ==
Upload the Below Post Title and Below Post Content Ads folder to the /wp-content/plugins/directory

Activate the plugin through the \'Plugins\' menu in WordPress
